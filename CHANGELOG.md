# 更新日志

## [1.3.0] - 2026-06-27

### 新增
- 群聊专属规则：支持为同一 QQ 号在不同群设置不同的注入文本
- 每个用户规则下可添加多条群聊覆盖规则（group_overrides）

### 变更
- `inject_prompt` 匹配逻辑：优先使用当前群聊的专属文本，找不到则使用默认文本
- 导入导出兼容 `group_overrides` 字段

## [1.2.0] - 2026-06-27

### 新增
- 每条规则支持启用/禁用开关，无需删除即可临时关闭

## [1.1.0] - 2026-06-26

### 新增
- `/tsc 导出规则`：导出当前全部规则为 JSON
- `/tsc 导入规则 [JSON]`：以 QQ 号合并导入规则（同名覆盖、新号追加）
- `/tsc help`：查看指令帮助
- 导入后自动调用 `save_config()` 持久化到磁盘

### 变更
- 代码结构优化：分离公共方法、注入逻辑、指令处理
- 配置由 AstrBot 统一管理（`data/config/`）

## [1.0.0] - 2026-06-25

### 新增
- 首次发布
- 根据 QQ 号自动向 LLM system_prompt 注入自定义规则
- WebUI 配置界面
